- CSS/JS: Không inline, đặt tại assets/css hoặc assets/js.
- Fonts: Font chữ đặt tại folder fonts.
- Tests: Vấn đề khó xử lý thì tạo file test tại folder tests.
- Logs: Ghi log đầy đủ vào folder logs.
- Media: Ảnh tĩnh tại folder img, ảnh tải lên tại folder uploads (lưu DB & folder).
- Database: Xây dựng qua migrations, tạo schema sau cùng.
- Code style: Không emoji/icon nhảm. Tuân thủ cấu trúc MVC & API (models, views, controllers, routes, api).
